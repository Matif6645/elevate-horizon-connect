# Elevate Horizon Connect

## Author

Atif

## Project Purpose

Elevate Horizon Connect is a mobile application developed as part of a TAFE assessment.  
The purpose of the app is to allow users to browse, view, and register for community events in a simple and user-friendly way.

## Target Audience

- Community members looking for local events
- Students and professionals interested in fitness, social, and networking events
- Users who want a simple event discovery experience on mobile

## Core Features

- Home screen with quick overview
- Events list screen displaying available events
- Event details screen showing time, location, and available spots
- Event registration screen (mock/demo)
- Settings screen (basic)
- Loading screen while data is fetched
- Event Not Found screen for invalid navigation

## Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript
- Git & GitHub for version control

## Data Source

- Event data is fetched from a hosted JSON file (mock data)

## Project Status

Core functionality completed.  
The app meets the minimum viable product (MVP) requirements and has been tested using Expo Go and a web emulator.

## How to Run the App

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
